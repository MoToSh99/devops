Følgende commands køres i terminalen;

For at hente golang:
    sudo apt install golang-go

For at hente vores web-framework:
    go get github.com/gorilla/mux
    go get github.com/gorilla/sessions


For at compile en .go fil:
    go build minitwit.go

For at køre en .go fil:
    go run minitwit.go

For at hente sqlite3 i go-form:
    go get github.com/mattn/go-sqlite3
